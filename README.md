# Switch Magazine

Switch Magazine is a lead-generation marketplace + editorial site focused on career-switchers, employers, and accredited training providers.

## Product Goals
- Help career switchers discover funded training programs and apprenticeships.
- Help employers source entry-level talent.
- Help schools grow enrollments and employer partnerships.
- Monetize through qualified leads, partner subscriptions, and featured placements.

## MVP Pages
- Homepage
- For Students
- For Employers
- For Schools
- Trade page template (`/trades/[slug]`)
- City page template (`/cities/[slug]`)
- Program page template (`/programs/[slug]`)
- Apply page (`/apply`)

## Core Tech Stack
- **Frontend**: Next.js (App Router)
- **Database/Auth**: Supabase
- **CMS (Guides/Editorial)**: Sanity
- **Hosting**: Vercel
- **Email**: Resend
- **CRM**: HubSpot
